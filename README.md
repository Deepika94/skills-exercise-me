# Skills Exercise: Core Learning Challenge

Practically explore a guided skills exercise that introduces core concepts, step-by-step practice, and a simple project outcome.

## Welcome

- **Who is this for**: learners preparing for technical practice exercises or completing a skills-based challenge.
- **What you'll learn**: how to approach a guided exercise, interpret requirements, and complete tasks in a structured learning environment.
- **What you'll build**: a polished exercise repository with clear goals, prerequisites, and a start guide.
- **Prerequisites**:
  - basic familiarity with GitHub repositories
  - a web browser and GitHub account

- **How long**: This exercise takes less than 20 minutes to review and begin.

In this exercise, you will:

1. review the learning objectives and project goals.
1. update repository content with exercise-specific details.
1. confirm the exercise is ready for sharing or use as a learning module.

### How to start this exercise

Copy the repository into your own GitHub account, wait a few seconds for the template setup to complete, then refresh the page.
